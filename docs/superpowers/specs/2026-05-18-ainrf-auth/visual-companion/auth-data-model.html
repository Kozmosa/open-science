多用户鉴权 — 数据模型

Section 1/5 · User 表 + 资源 ownership + Project Collaborators + Environment 权限组

1. User 表（新 SQLite DB: auth.sqlite3）

CREATE TABLE users (
  id TEXT PRIMARY KEY,
  username TEXT NOT NULL UNIQUE,
  password_hash TEXT NOT NULL,  -- bcrypt
  display_name TEXT NOT NULL,
  role TEXT NOT NULL DEFAULT 'member',  -- 'admin' | 'member'
  status TEXT NOT NULL DEFAULT 'pending',  -- 'pending' | 'active' | 'disabled'
  created_at TEXT NOT NULL,
  activated_at TEXT,  -- Admin 审批后设置
  last_login_at TEXT
);

-- JWT refresh tokens（支持 logout）
CREATE TABLE refresh_tokens (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL REFERENCES users(id),
  token_hash TEXT NOT NULL,
  expires_at TEXT NOT NULL,
  created_at TEXT NOT NULL
);

2. 资源表扩展 — user ownership

资源 新增列 说明
task_harness_tasks owner_user_id TEXT 创建 task 的用户
projects owner_user_id TEXT 创建 project 的用户
workspaces owner_user_id TEXT workspace 所属用户
task_sessions owner_user_id TEXT session 创建者
user_environment_bindings （已有 user_id） 无需改动
所有 owner_user_id 用 ALTER TABLE ADD COLUMN，默认 NULL。Admin 用户（PI）看到所有资源（不按 owner 过滤）。

3. Project Collaborators（新表）

CREATE TABLE project_collaborators (
  project_id TEXT NOT NULL,
  user_id TEXT NOT NULL,
  role TEXT NOT NULL DEFAULT 'member',  -- 'member' | 'viewer'
  added_by_user_id TEXT NOT NULL,
  added_at TEXT NOT NULL,
  PRIMARY KEY (project_id, user_id)
);
member: 可读写 project 下所有 tasks/sessions/workspace。
viewer: 只读（可查看 task 详情/output，不能创建/修改/取消）。

4. Environment 权限组（新表）

CREATE TABLE environment_access (
  environment_id TEXT NOT NULL,
  user_id TEXT NOT NULL,
  max_concurrent_tasks INTEGER,  -- NULL = 无限制
  granted_by_user_id TEXT NOT NULL,
  granted_at TEXT NOT NULL,
  PRIMARY KEY (environment_id, user_id)
);

-- Admin 可看到所有 env；普通用户只看到 environment_access 中自己的行
-- "全局可用"的 env：Admin 为所有用户各插一行

5. 权限检查逻辑（伪代码）

Admin (role='admin'): 所有资源 CRUD 全通

Project 访问:
  is_admin OR
  project.owner_user_id == current_user.id OR
  project_collaborators 中存在 (project_id, user_id)

Task 访问:
  is_admin OR
  task.owner_user_id == current_user.id OR
  该 task 的 project 的 collaborator

Environment 访问:
  is_admin OR
  environment_access 中存在 (environment_id, user_id)

Workspace 访问:
  is_admin OR
  workspace.owner_user_id == current_user.id OR
  该 workspace 的 project 的 collaborator
