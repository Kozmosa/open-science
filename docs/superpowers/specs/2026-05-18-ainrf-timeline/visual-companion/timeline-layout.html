关键设计决策

决策 选择 理由
时间轴单位 自适应（按数据跨度选择 hour/day/week） 跨天时用天，当天内用小时，跨周用周
同一 attempt 的多行 同一 session 的 attempts 在同一行，多个色块 对齐 humanize2 的 Gantt per-session 模式
时间范围选择 Quick presets (Today/7d/30d) + 日期范围 picker 常用场景覆盖，自定义需求用 range picker
Tooltip 交互 Hover 显示 attempt 详情卡片（duration, cost, status, task link） 不增加视觉负担，按需查看
数据获取 一次加载所有 sessions + 按需加载 attempt 详情 减少 API 调用；sessions 列表轻量，attempts 按需
