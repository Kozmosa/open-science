| 功能 | 说明 |
|------|------|
| [项目管理](/scholar-agent/projects/) | ReactFlow Canvas DAG 可视化、任务创建与连线 |

| [工作区](/scholar-agent/workspace/) | 目录管理、文件浏览器、Monaco 编辑器 |

| [资源监控](/scholar-agent/resources/) | GPU/CPU/内存实时监控、进程树 |
| [系统设置](/scholar-agent/settings/) | 用户偏好、LLM Provider、技能仓库、Admin 面板 |
