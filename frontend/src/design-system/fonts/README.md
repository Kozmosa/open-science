# Noto Sans osci font assets

The four Latin-only WOFF2 files are provided by the exact npm dependency
`@fontsource/noto-sans@5.2.10` and are bundled locally by Vite. The upstream
font source is the Noto Project's Latin, Greek and Cyrillic repository:
`https://github.com/notofonts/latin-greek-cyrillic`.
